A Conhex-like game.

Dependencies:
 * A C++11 compiler.
 * fbxsdk: Copy the lib and header files into the 'lib' directory.
 * glfw
 * GLEW
 * GL
 * OpenGL version?

This game is aimed to run on Linux and Windows machines.
